Optimisation & Deep Learning

The main objective of this assignment is to find an optimal model for a problem for which historical data is given in a dataset. This is an end to end project that addresses optimality in hyper parameter selection and model parameter selection using different models.

The dataset used from a chicago crime database and was thoroughly preprocessed before a exploratory analysis was carried out. The prediction models were selected and tuning and parameters were selected. I chose decision trees and created models before and after tuning to determine difference between both values. Overall, Decision Trees as compared to other models selected by group members was determined to be more accurate.
